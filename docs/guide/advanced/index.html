Advanced Guide

Advanced topics are public features, but they are not the normal starting point for a XIS application. Use them when the application shape requires distribution, custom browser behavior, framework extension, or infrastructure integration.

Microfrontend architectureDistributed pages and frontlets with one coherent browser application. Reusable web artifactsLibrary-provided controllers, replaceable templates, and reusable UI behavior. AspectsNative-image-friendly advice for framework and infrastructure concerns. Custom proxiesGenerated clients, repositories, and infrastructure extensions. SQL transactionsExplicit transaction control for cases where repository operations need a shared boundary. Custom JavaScriptExtension files, EL functions, and browser-side integrations. Integration-test browserThe server-side browser model for testing rendered pages. ThemeThe optional XIS theme and generated standard UI structure.
