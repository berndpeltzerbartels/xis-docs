Index

This page is the lookup index for the public XIS API surface. Use the alphabetical list when you know the name, and the thematic list when you know the problem area but not the annotation or interface yet.

By topic

Pages and UI composition

  • @Page, @WelcomePage, @Frontlet, @Modal
  • @Include, @Router, @Route, @Title
  • PageResponse, PageUrlResponse, FrontletResponse, ModalResponse

Actions, model data, and forms

  • @ModelData, @Action, @ActionParameter, @FormData
  • @SharedValue, @ClientState, @LocalStorage, @SessionStorage
  • @Mandatory, @MinLength, @RegExpr, @Validate, Validator

Dependency injection and extension points

  • @Component, @Service, @Bean, @Inject, @Init
  • Advice, AdviceInvocation, @UseAdvice
  • ProxyFactory, @Proxy

Persistence

  • @Entity, @Column, @JsonColumn, @NoColumn
  • CrudRepository, @Repository, @Select, @Transactional
  • DDL, Table, Column, PrimaryKey
  • @MongoDocument, MongoCrudRepository, @MongoRepository, @MongoWatch

Security, users, and refresh events

  • @Roles, @OwnedBy, @Authenticated
  • @UserId, @ClientId, @BearerToken
  • RefreshEventPublisher, @RefreshOnUpdateEvents, @EventListener

Plain HTTP, tests, and operations

  • HttpRequest, HttpResponse, HttpFilter, FilterChain, ResponseEntity
  • @Get, @Post, @Put, @Delete, @Produces
  • @XisBootTest, @Mock, @Spy, @InTestContext
  • @XISBootApplication, @XISHttpApplication, @Scheduled
