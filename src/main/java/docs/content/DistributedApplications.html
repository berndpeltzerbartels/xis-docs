Work in Progress: Distributed application support is currently under development and not yet available in XIS. This feature will allow widgets to be loaded from remote servers, enabling true micro-frontend architectures. This chapter will be completed once the feature is implemented.

XIS will support distributed application architectures where widgets can be developed, deployed, and served independently from different servers. This will enable micro-frontend patterns and team autonomy.

Planned Features
  • Widget Servers: Deploy widgets independently on separate servers
  • Remote Widget Loading: Load widgets from different origins at runtime
  • Cross-Server Communication: Secure communication between distributed components
  • Deployment Patterns: Best practices for distributed XIS applications

Detailed documentation will be added once this feature is released.
