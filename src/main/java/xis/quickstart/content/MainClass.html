Now let's create the entry point for your Spring Boot application.

Create the package structure

Create the following directory structure:

src/main/java/quickstart/
Create the Application class

Create a file Application.java in the quickstart package: 

package quickstart;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class Application {
    
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}

This is a standard Spring Boot application class. The @SpringBootApplication annotation enables auto-configuration and component scanning.

XIS integrates seamlessly with Spring Boot - no additional configuration is needed. The XIS components will be automatically discovered and configured.
