This is the quickstart guide for XIS with Spring Boot. It will help you set up your first XIS project and build a simple web application.

Prerequisites
  • Java 17 or higher
  • Gradle 8.0 or higher
  • Your favorite IDE (IntelliJ IDEA, Eclipse, VS Code)
Create a new project

Create a new directory for your project and add the following files:

build.gradle
plugins {
    id 'java'
    id 'org.springframework.boot' version '3.3.0'
    id 'io.spring.dependency-management' version '1.1.5'
    id 'one.xis.plugin' version '${xisVersion}'
}

group = 'quickstart'
version = '0.0.1-SNAPSHOT'

repositories {
    mavenCentral()
}

dependencies {
    // XIS libraries
    implementation('one.xis:xis-spring')
    testImplementation('one.xis:xis-test')

    // Lombok (optional, but recommended)
    compileOnly 'org.projectlombok:lombok:1.18.30'
    annotationProcessor 'org.projectlombok:lombok:1.18.30'

    // Spring Boot
    implementation 'org.springframework.boot:spring-boot-starter-web'

    // JUnit 5 for testing
    testImplementation 'org.springframework.boot:spring-boot-starter-test'
}

tasks.withType(JavaCompile) {
    options.encoding = 'UTF-8'
}

test {
    useJUnitPlatform()
}
settings.gradle
rootProject.name = 'my-xis-app'

That's it! Your project is now ready. Continue to the next section to create your first page.
