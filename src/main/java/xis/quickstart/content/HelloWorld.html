Let's create your first XIS page - a simple "Hello World" application.

Create the Page controller

Create a subpackage helloworld inside quickstart and add a file HelloWorld.java: 

package quickstart.helloworld;

import one.xis.ModelData;
import one.xis.Page;

@Page("/hello-world.html")
class HelloWorld {

    @ModelData
    String greeting() {
        return "Hello, World!";
    }
}
Generate the HTML template

XIS can automatically generate missing HTML templates for your pages and widgets. You have two options:

Option 1: IDE

Reload/refresh your Gradle project in your IDE, then execute the xis/templates task from the Gradle tool window.

Option 2: Command Line

First, create a Gradle wrapper if you don't have one yet:

gradle wrapper --gradle-version 8.12

Then run the templates task:

./gradlew templates

This will generate a file HelloWorld.html in the same package as your Java class.

Edit the HTML template

Open the generated HelloWorld.html file and replace its content with: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8"/>
    <title>${greeting}</title>
</head>
<body>
<h1>${greeting}</h1>
</body>
</html>

The ${greeting} expression will be replaced with the value from the greeting() method in your controller at runtime.

Your project structure

After these steps, your project should look like this:

src/
└── main/
    └── java/
        └── quickstart/
            ├── Application.java
            └── helloworld/
                ├── HelloWorld.java
                └── HelloWorld.html
