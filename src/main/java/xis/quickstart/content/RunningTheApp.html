Now it's time to start your application and see your Hello World page in action!

Start the application

Run the following command in your project directory:

./gradlew bootRun

Or if you haven't created the Gradle wrapper yet, use: 

gradle bootRun

You should see Spring Boot starting up. Wait for a message like: 

Started Application in 2.5 seconds
Open in browser

Once the application is running, open your web browser and navigate to: 

http://localhost:8080/hello-world.html

You should see a page displaying: 

Hello, World!

Congratulations! 🎉 You've successfully created and run your first XIS application.

Stop the application

To stop the application, press Ctrl+C in the terminal where it's running.
