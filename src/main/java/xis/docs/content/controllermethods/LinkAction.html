Actions can be triggered by links using the xis:action attribute. This allows you to execute controller methods when users click on links, without requiring buttons or forms.

Basic Link Action

Use xis:action on an <a> tag to trigger a controller method:

@Action
void markAsRead() {
    message.read = true;
    messageService.update(message);
}

Template:

<a xis:action="markAsRead">Mark as read</a>
Link Action with Parameters

Pass parameters to the action method using <xis:parameter> tags:

@Action
void deleteItem(@ActionParameter("itemId") String itemId) {
    itemService.delete(itemId);
}

Template:

<a xis:action="deleteItem">
    <xis:parameter name="itemId" value="${item.id}"/>
    Delete
</a>
