XIS Boot is the standalone runtime for XIS applications, providing a lightweight embedded server without requiring Spring Boot, Micronaut, or other application frameworks. It's perfect for:

  • Rapid prototyping and development
  • Simple web applications without complex framework requirements
  • Learning XIS without framework overhead
  • Microservices with minimal dependencies

Quick Start

A minimal XIS Boot application requires just a main class and your page/widget controllers:

public class Application {
    public static void main(String[] args) {
        XisApplication.run(Application.class, args);
    }
}

@Page("/index.html")
class IndexPage {
    @ModelData("message")
    String getMessage() {
        return "Hello XIS Boot!";
    }
}

XIS Boot automatically discovers your pages and widgets in the same package and starts an embedded web server.
