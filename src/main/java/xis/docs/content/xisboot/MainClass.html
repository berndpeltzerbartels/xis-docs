The main class is the entry point of your XIS Boot application. It bootstraps the XIS runtime and starts the embedded web server.

Basic Main Class

Create a simple main class that starts XIS Boot:

package com.example.myapp;

import one.xis.boot.XISBootRunner;

public class Application {
    public static void main(String[] args) {
        XISBootRunner.run(Application.class, args);
    }
}

The XisApplication.run() method:

  • Scans packages in this package or its subpackages (like SpringBoot)
  • Registers all controllers and components
  • Starts the embedded web server (default port 8080)
  • Processes the port number from command-line arguments if provided
Application Structure

Organize your application with a clear package structure:

com.example.myapp/
├── Application.java         # Main class
├── pages/
│   ├── HomePage.java
│   └── HomePage.html
└── widgets/
    ├── HeaderWidget.java
    └── HeaderWidget.html
Specifying the Port

You can specify the server port by passing it as a command-line argument:

java -jar myapp.jar 9090

If no port is specified, the application will use the default port 8080.
