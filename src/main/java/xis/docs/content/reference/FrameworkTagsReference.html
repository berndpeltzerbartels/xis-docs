Quick reference of available XIS framework tags for building dynamic templates. For detailed information see Framework Tags Documentation.

Core Control Flow Tags
Tag Attributes Description
<xis:foreach> var, array
or xis:foreach="var:array"		 Iterates over collections to repeat content for each item
<xis:if> condition
or xis:if="expression"		 Conditionally renders content based on expressions
<xis:widget-container> container-id (required), default-widget, scroll-to-top Creates a container for dynamic widget loading
<xis:widget> name (required) Short form of widget container when no container-id is needed
<xis:message> message-for (required) Displays validation messages for form fields
<xis:parameter> name (required), value (required) Passes parameters to actions and widgets
Form Elements
Tag Attributes Description
<xis:form> binding (required), action XIS form element normalized to standard HTML form
<xis:input> binding (required), type, standard HTML input attributes XIS input element normalized to HTML input
<xis:select> binding (required), standard HTML select attributes XIS select element normalized to HTML select
<xis:checkbox> binding (required), value, standard HTML input attributes XIS checkbox element normalized to HTML input type="checkbox"
<xis:radio> binding (required), value (required), name XIS radio button element normalized to HTML input type="radio"
<xis:button> action, type, standard HTML button attributes XIS button element normalized to HTML button
<xis:submit> standard HTML input attributes XIS submit button element normalized to HTML submit input
Navigation Elements
Tag Attributes Description
<xis:a> page OR widget (required), target-container, parameters, refresh XIS link element for page navigation or widget loading
<xis:action> action (required), target-container XIS action link element for triggering controller actions
Special Purpose Tags
Tag Attributes Description
<xis:template> xmlns:xis (required) Wrapper tag for widget templates with multiple root elements
<xis:raw> text (optional, default: false) Outputs content without XIS processing (HTML or text mode)
