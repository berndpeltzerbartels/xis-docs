Quick reference of available XIS framework attributes that can be used as alternatives to tags. For detailed information see Framework Attributes Documentation.

Iteration Attributes
Attribute Syntax Description
xis:foreach xis:foreach="var:array" Repeats content for each item in a collection
xis:repeat xis:repeat="var:array" Repeats the element itself for each item in a collection
Conditional Attributes
Attribute Syntax Description
xis:if xis:if="expression" Conditionally renders element based on expression
Widget Container Attributes
Attribute Syntax Description
xis:widget-container xis:widget-container="containerId" Creates a container for dynamic widget loading
xis:default-widget xis:default-widget="WidgetId" Specifies default widget to load in container
xis:scroll-to-top xis:scroll-to-top="true" Scrolls to top when widget loads
Form & Data Binding Attributes
Attribute Syntax Description
xis:binding xis:binding="fieldName" Binds element to data model for automatic data binding
xis:message-for xis:message-for="fieldName" Displays validation messages for specific field
xis:global-messages xis:global-messages="true" Displays all global validation messages
xis:error-class xis:error-class="className" Applies CSS class when bound field has validation errors
Action & Navigation Attributes
Attribute Syntax Description
xis:action xis:action="actionName" Triggers controller action when clicked
xis:page xis:page="/page.html" Navigates to specific page when clicked (only on <a> elements)
xis:widget xis:widget="WidgetId" Loads widget into surrounding container (only on <a> elements)
xis:target-container xis:target-container="containerId" Specifies target container for widget/action result
xis:parameters xis:parameters="key1=val1&key2=val2" Passes query parameters to page navigation
xis:refresh xis:refresh="false" Controls whether page refresh happens on navigation
xis:include xis:include="key" Includes reusable template fragment from @Include bean
Selection Attributes
Attribute Syntax Description
xis:selection-group xis:selection-group Marks container as selection group
xis:selection-class xis:selection-class="className" CSS class applied to selected items in group
