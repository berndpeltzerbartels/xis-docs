Quick reference of available class-level annotations from the xis-controller-api package. For detailed information see Class Annotations Documentation.

Qualified Name Simple Name Description
one.xis.Page @Page Marks a class as a page controller with a specific URL path
one.xis.Widget @Widget Marks a class as a reusable widget component
one.xis.WelcomePage @WelcomePage Designates a page as the default landing page of the application
one.xis.RefreshOnUpdateEvents @RefreshOnUpdateEvents Widget automatically reloads when specific events are emitted
one.xis.auth.Roles @Roles Defines required roles for accessing the class
one.xis.Include @Include Marks a bean for inclusion in templates via <xis:include> tag
