Quick reference of available field and validation annotations from the xis-controller-api package. For detailed information see Validation Documentation.

Validation Annotations
Qualified Name Simple Name Description
one.xis.Mandatory @Mandatory Validates that value is not null, empty, or whitespace
one.xis.Email @Email Validates that string is a valid email format
one.xis.MinLength @MinLength Validates minimum length for strings or collections
one.xis.AllElementsMandatory @AllElementsMandatory Validates that all collection elements are non-null
one.xis.LabelKey @LabelKey Provides context-specific field labels in error messages
one.xis.UseFormatter @UseFormatter Specifies custom formatter for parsing field values
one.xis.NullAllowed @NullAllowed Explicitly allows null values for optional fields
