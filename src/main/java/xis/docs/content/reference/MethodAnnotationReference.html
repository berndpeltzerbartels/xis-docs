Quick reference of available method-level annotations from the xis-controller-api package. For detailed information see Method Annotations Documentation.

Qualified Name Simple Name Description
one.xis.ModelData @ModelData Provides data to templates, making return value available under method name or custom key
one.xis.Action @Action Handles user interactions like button clicks, link navigation, or form submissions
one.xis.FormData @FormData Binds form data to method for automatic type conversion and validation
one.xis.SharedValue @SharedValue Caches method return value in request scope for reuse by other methods
one.xis.Title @Title Sets the page or widget title from method return value
one.xis.LocalStorage @LocalStorage Provides data from browser's localStorage
one.xis.SessionStorage @SessionStorage Provides data from browser's sessionStorage
one.xis.GlobalVariable @GlobalVariable Stores/retrieves data from client-side global variables (not persistent across page reloads)
one.xis.HtmlFile @HtmlFile Specifies custom location for the HTML template file
one.xis.auth.Roles @Roles Defines required roles for accessing the method
