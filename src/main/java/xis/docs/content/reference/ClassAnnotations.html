Kurze Übersicht der verfügbaren Class Annotations aus dem xis-controller-api Package. Für detaillierte Informationen siehe Class Annotations Dokumentation.

Qualified Name Simple Name Beschreibung
one.xis.Page @Page Markiert eine Klasse als Page-Controller mit spezifischem URL-Pfad
one.xis.Widget @Widget Markiert eine Klasse als wiederverwendbares Widget
one.xis.WelcomePage @WelcomePage Markiert eine Page als Standard-Startseite der Anwendung
one.xis.RefreshOnUpdateEvents @RefreshOnUpdateEvents Widget wird automatisch aktualisiert wenn bestimmte Events ausgelöst werden
one.xis.context.Component @Component Markiert eine Klasse als Component für Dependency Injection
one.xis.context.DefaultComponent @DefaultComponent Markiert eine Standard-Implementierung die überschrieben werden kann
one.xis.auth.Roles @Roles Definiert erforderliche Rollen für Zugriff auf die Klasse
