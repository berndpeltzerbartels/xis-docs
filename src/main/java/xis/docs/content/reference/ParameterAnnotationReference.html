Quick reference of available parameter annotations from the xis-controller-api package. For detailed information see Parameter Annotations Documentation.

Qualified Name Simple Name Description
one.xis.QueryParameter @QueryParameter Injects URL query parameters from requests
one.xis.PathVariable @PathVariable Injects URL path variables from route patterns
one.xis.ActionParameter @ActionParameter Injects parameters passed from template actions
one.xis.WidgetParameter @WidgetParameter Injects parameters passed to widgets (widget-only)
one.xis.FormData @FormData Injects form data bound from HTML forms
one.xis.LocalStorage @LocalStorage Injects data from browser's localStorage (persistent)
one.xis.SessionStorage @SessionStorage Injects data from browser's sessionStorage (tab-specific)
one.xis.GlobalVariable @GlobalVariable Injects data from client-side global variables (not visible in browser tools)
one.xis.SharedValue @SharedValue Injects cached values from methods within the same request
one.xis.ClientId @ClientId Injects unique client identifier (stored in localStorage)
one.xis.ClientStorage @ClientStorage Injects data from server-side client storage
one.xis.auth.UserId @UserId Injects user identifier of logged-in user (requires xis-authentication)
