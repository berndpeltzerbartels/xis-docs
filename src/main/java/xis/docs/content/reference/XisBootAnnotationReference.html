Quick reference of available annotations from the xis-context package used in XIS Boot applications. For detailed information see Context Documentation.

Class-Level Annotations
Qualified Name Simple Name Description
one.xis.context.Component @Component Marks a class as a component for dependency injection
one.xis.context.DefaultComponent @DefaultComponent Marks a default implementation that can be overridden
one.xis.context.Bean @Bean Marks a method that produces a bean instance for dependency injection
one.xis.context.Service @Service Specialized component annotation for service layer classes
one.xis.context.EventListener @EventListener Marks a method as an event listener for application events emitted by EventEmitter
Field-Level Annotations
Qualified Name Simple Name Description
one.xis.context.Inject @Inject Injects a dependency into a field or constructor parameter
Method-Level Annotations
Qualified Name Simple Name Description
one.xis.context.Init @Init Marks a method to be executed after dependency injection is complete
one.xis.context.Bean @Bean Marks a method that produces a bean instance for dependency injection
one.xis.context.EventListener @EventListener Marks a method as an event listener for application events emitted by EventEmitter
