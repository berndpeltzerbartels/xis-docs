XIS provides a simple mechanism to highlight selected elements in lists, menus, or any grouped content using the xis:selection-class and xis:selection-group attributes.

Basic Usage

To make elements selectable with visual highlighting:

  1. Wrap the selectable elements in a container with xis:selection-group
  2. Add xis:selection-class="className" to each selectable element
  3. Define the CSS class that represents the selected state

Example: List Selection

<ul xis:selection-group>
    <li xis:selection-class="selected" id="item1">Item 1</li>
    <li xis:selection-class="selected" id="item2">Item 2</li>
    <li xis:selection-class="selected" id="item3">Item 3</li>
</ul>

CSS for highlighting:

.selected {
    background-color: #007bff;
    color: white;
    font-weight: bold;
}

How It Works

When a user clicks on an element with xis:selection-class:

  • The specified CSS class is added to the clicked element
  • The CSS class is removed from all other elements in the same selection group
  • Only one element per group can have the selection class at a time

Menu Navigation Example

<nav xis:selection-group>
    <a href="#home" xis:selection-class="active">Home</a>
    <a href="#about" xis:selection-class="active">About</a>
    <a href="#contact" xis:selection-class="active">Contact</a>
</nav>
nav a.active {
    border-bottom: 3px solid #007bff;
    color: #007bff;
}

Multiple Selection Groups

You can have multiple independent selection groups on the same page: 

<!-- First group -->
<div xis:selection-group>
    <button xis:selection-class="selected">Option A</button>
    <button xis:selection-class="selected">Option B</button>
</div>

<!-- Second group -->
<div xis:selection-group>
    <button xis:selection-class="highlighted">Choice 1</button>
    <button xis:selection-class="highlighted">Choice 2</button>
</div>

Use Cases

  • Navigation menus: Highlight the current page/section
  • Tab interfaces: Show active tab with visual feedback
  • List selection: Allow users to select items from a list
  • Dashboard tiles: Highlight selected dashboard sections
  • Image galleries: Show which image is currently selected
