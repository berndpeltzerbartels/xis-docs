XIS-Theme is an optional dependency that provides a clean, professional design system for rapid prototyping and production-ready applications. It ensures your XIS applications look professional right from the start, without requiring designer involvement.

Adding XIS-Theme

Add the dependency to your build.gradle:

dependencies {
    implementation 'one.xis:xis-theme'
}

The CSS and XIS logo are automatically included in your application. Simply reference xis.css in your HTML templates:

<link rel="stylesheet" href="xis.css">
What's Included
  • Professional Typography: Clean, readable fonts with proper sizing and spacing
  • XIS Logo: SVG logo included and ready to use
  • Navigation Component: Responsive navbar with dropdown support
  • Grid System: Flexible 2-11 column grid layout system
  • Form Controls: Styled inputs, textareas, selects, buttons
  • Error Handling: Form validation error styles and toast notifications
  • Utility Components: Panels (warning, tip, message), code blocks, sidebar
  • CSS Variables: Easy customization via CSS custom properties
Grid System

XIS-Theme provides a flexible grid system with classes .col2 through .col11:

<div class="col3">
    <div>Column 1</div>
    <div>Column 2</div>
    <div>Column 3</div>
</div>

Spanning columns: Use .span1 through .span9 to span multiple columns:

<div class="col4">
    <div class="span2">Spans 2 columns</div>
    <div>Single column</div>
    <div>Single column</div>
</div>
Form Layout Example

Forms automatically align within the grid:

<form>
    <div class="col2">
        <div class="span1">
            <label>First Name</label>
            <input type="text" name="firstName" value="${user.firstName}"/>
            <div class="form-error" xis:if="${errors['user.firstName']}">
                ${errors['user.firstName']}
            </div>
        </div>
        
        <div class="span1">
            <label>Last Name</label>
            <input type="text" name="lastName" value="${user.lastName}"/>
            <div class="form-error" xis:if="${errors['user.lastName']}">
                ${errors['user.lastName']}
            </div>
        </div>
        
        <div class="span2">
            <input type="submit" value="Save"/>
        </div>
    </div>
</form>

Error Display

Field-level and global error messages are styled automatically:

<!-- Field-level error -->
<div class="form-field error">
    <label>Email</label>
    <input type="email" value="${user.email}"/>
    <div class="form-error">${errors['user.email']}</div>
</div>

<!-- Global form error -->
<div class="form-error-global" xis:if="${hasErrors}">
    Please correct the errors above.
</div>
Toast Notifications

Fixed-position toast notifications for user feedback:

<div class="toast-container">
    <div class="toast error">An error occurred</div>
    <div class="toast success">Changes saved successfully</div>

</div>

Customization

Customize the theme by overriding CSS variables in your own stylesheet:

:root {
    --accent: #007bff;        /* Primary brand color */
    --base-font: 16px;        /* Base font size */
    --grid-gap: 20px;         /* Grid spacing */
    --radius: 6px;            /* Border radius */
    --ctrl-h: 44px;           /* Form control height */
}
Responsive Design

All components are mobile-responsive out of the box:

  • Navigation collapses on mobile devices
  • Grid columns stack on smaller screens
  • Form controls adapt to screen width
  • Touch-friendly target sizes

When to Use XIS-Theme
  • Prototyping: Get professional-looking prototypes without design work
  • Internal Tools: Build admin panels and dashboards quickly
  • MVP Development: Launch with a solid foundation
  • Small Projects: Simple applications that don't require custom branding

Note: XIS-Theme is designed to be replaced with your own design system when needed. It provides a solid starting point that can either remain in production or serve as a template for your custom styling.
