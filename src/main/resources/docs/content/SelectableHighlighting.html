SPA-Menüpunkt-Hervorhebung

Dieses Kapitel beschreibt, wie Menüpunkte in einer Single Page Application (SPA) hervorgehoben werden können, um die aktuelle Auswahl für den Benutzer sichtbar zu machen.

Beispiel für CSS-Klassen

.selectable {
    cursor: pointer;
}
.selected {
    background: #e0e0e0;
    font-weight: bold;
    border-radius: 4px;
}

Typische Implementierung

Die Hervorhebung erfolgt meist durch das Setzen einer selected-Klasse auf das aktuell aktive Menüelement. Dies kann serverseitig oder per JavaScript erfolgen.

Integration in XIS

  • Im HTML-Template: <li class="selectable ${selected ? 'selected' : ''}">Menüpunkt</li>
  • Im Controller: Modellwert selected setzen
  • Optional: JavaScript für dynamische Auswahl
