Coverage checklist

This page guards completeness. Nothing that is currently described in the XIS README or module READMEs should vanish from the static documentation.

Must be covered

The checklist is intentionally detailed. It mirrors the current README and user-doc map so the new static site can replace the README collection without losing public behavior.

Current completion status. The conceptual chapters explain the framework in book form. The Complete Guide contains the detailed public API material, including annotations, tags, validation, persistence, security, Gradle, native images, tests, and advanced extension topics.
